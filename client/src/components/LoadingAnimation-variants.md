# Варианты корабликов для анимации загрузки

## Вариант 1: Текущий (Парусник с одним парусом)
- Классический парусник
- Один треугольный парус
- Красный флаг
- Коричневый корпус

## Вариант 2: Яхта (Современная яхта)
- Современный дизайн
- Два паруса
- Более изящный корпус
- Синий/белый цвет

## Вариант 3: Катер (Моторная лодка)
- Без парусов
- Современный дизайн
- Кабина
- Синий корпус

## Вариант 4: Парусник с двумя парусами
- Два паруса (передний и задний)
- Классический вид
- Более детализированный

## Вариант 5: Простой кораблик (Минималистичный)
- Простой силуэт
- Минимум деталей
- Чистый дизайн

## Вариант 6: Круизный лайнер
- Большой корабль
- Много окон
- Современный вид
