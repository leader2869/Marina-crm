# Установка GitHub CLI на macOS

## Способ 1: Через официальный установщик (рекомендуется)

1. Откройте браузер и перейдите на: https://cli.github.com/
2. Скачайте установщик для macOS (Apple Silicon для M1/M2/M3)
3. Откройте скачанный файл `.pkg`
4. Следуйте инструкциям установщика
5. После установки проверьте:
```bash
gh --version
```

## Способ 2: Через Homebrew (если установлен)

```bash
brew install gh
```

## Настройка аутентификации

После установки выполните:

```bash
gh auth login
```

Следуйте инструкциям:
1. Выберите `GitHub.com`
2. Выберите `HTTPS` для протокола Git
3. Выберите `Login with a web browser` (Войти через веб-браузер)
4. Нажмите Enter, скопируйте код из терминала
5. Откройте ссылку в браузере, вставьте код и авторизуйтесь

## Проверка аутентификации

```bash
gh auth status
```

## После настройки

Вы сможете использовать `git push` без ввода пароля, так как GitHub CLI будет автоматически обрабатывать аутентификацию.
